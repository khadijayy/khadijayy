### 🚀 Featured Projects

#### 🧩 [Synapse](https://github.com/khadijayy) — AI-Powered Team Formation & Mentor-Matching Platform
*Final Year Project · FastAPI · PostgreSQL · Flutter · Supabase · Firebase*

A platform that matches students into teams and mentors using a **seven-signal heuristic composite scoring engine** — combining Big Five/DISC personality compatibility, MMR-based diversity re-ranking, epsilon-greedy wildcard injection (so matches aren't purely deterministic), and a **Random Forest classifier** that predicts collaboration success.

- Backend: FastAPI + PostgreSQL on Render, with Supabase for auth/data and Firebase FCM for real-time notifications
- Frontend: Flutter/Dart (Riverpod, GoRouter), deployed on Netlify
- Built with FYP teammate Roumaisa Tanveer, supervised by Dr. Asif Sohail

#### 🤖 [AskRAG](https://github.com/khadijayy) — RAG-Based Codebase Q&A Tool
*Self-initiated · FastAPI · FAISS · Sentence Transformers · Gemini 2.5 Flash*

Ask natural-language questions about any codebase and get answers grounded in the actual source — not hallucinated. Connects via **GitHub OAuth**, indexes repos with **FAISS** + Sentence Transformers embeddings, and generates answers using **Gemini 2.5 Flash** via the `google-genai` SDK.

- Full retrieval-augmented generation pipeline, deployed and live
- PostgreSQL/Supabase for persistence
- Actively extending toward agentic tool-use (not just retrieval — taking actions on the codebase)

#### 💊 OnTime Meds — Medication Reminder App
*Android · Java · Firebase*

A simple, reliable reminder app focused on accessibility and ease of use for medication adherence.
